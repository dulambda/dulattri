Коротко о том, что реализовано (сухо, без лишнего): Полноэкранное меню (.fullmenu) — position: fixed, display: none по умолчанию, открывается/закрывается по кнопкам. Секция слайдера — верстка одного слайда + базовый JS для переключения слайдов (готово к дублированию). Вертикальный аккордеон — скрытая часть (.acc-body) спрятана через height: 0; overflow: hidden; и открывается динамически (анимация высоты). Слайдшоу (виджет): список контента и список аватаров — клик по аватару показывает соответствующий контент. Форма заказа: проверка полей, отправка POST на https://webdev-api.loftschool.com/sendmail с JSON, заголовком content-type: application/json. Модальное окно: появляется в зависимости от ответа сервера (сервер эмулирует случайный успех/ошибку). Показано сообщение из ответа. Есть закрытие по кнопке и клику вне модального окна.
